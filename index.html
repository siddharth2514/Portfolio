  • AI Attendance System


    Project Overview: In collaboration with a team, we developed an AI-powered attendance system that leverages facial recognition technology to automate and streamline the student attendance process. This project addresses inefficiencies in traditional roll-call methods by ensuring accurate and proxy-free attendance tracking.


    Technology Used:

  • Deep Learning Models: Utilized pre-trained face recognition models (e.g., ArcFace) for high-accuracy face detection and verification.
  • Computer Vision: Applied face alignment and preprocessing techniques to handle variations in lighting and orientation.
  • Database Integration: Stored student face embeddings and attendance records in a secure database (e.g., PostgreSQL or MongoDB).
  • Backend Framework: Implemented using Flask for handling registration, verification, and attendance report generation.


    • Functionality:

  • Teachers capture a class photo after each lecture.
  • The system automatically detects and verifies student faces against the registered database.
  • Attendance is marked instantly with minimal manual intervention.
  • Generates editable attendance reports to handle edge cases.


    • Project Significance: This project demonstrates my ability to integrate AI and computer vision to solve practical problems in the education sector. It reduces proxy attendance, saves valuable lecture time, and showcases the application of cutting-edge facial recognition technologies in a real-world scenario.

AI Career Counselor


Project Overview: In collaboration with a team, we developed an AI-powered Career Counseling system designed to guide students in making informed academic and career decisions. This project leverages advanced NLP, psychometric profiling, and real-time data analysis to deliver personalized career recommendations based on individual interests, strengths, and personality traits.


Technology Used:

  • Natural Language Processing (NLP): Used to understand user inputs and engage in dynamic, conversational interactions.
  • Psycho metric Analysis: Embedded stealth MBTI-style assessments to identify user personality types without explicit testing.
  • Database Integration: Stored user profiles, trait scores, and recommendation history using Firebase Firestore for real-time updates.
  • AI Recommendation Engine: Mapped user profiles to relevant career paths, learning resources, and top institutions.
  • Frontend & Voice Interaction: Integrated a user-friendly web interface with voice input/output for an interactive counseling experience.


    • Functionality:

  • Engages students in natural, emotionally supportive conversations to uncover their skills, interests, and personality traits.
  • Calculates a comprehensive psychometric profile in the background.
  • Provides tailored career suggestions, learning roadmaps, and college recommendations.
  • Tracks user progress and updates recommendations as they grow.

    • Project Significance: This project showcases my ability to apply AI and psychometric analysis to solve real-world guidance challenges in education. It empowers students with personalized, data-driven insights, bridging the gap between their current skills and future aspirations.

    Football Match Prediction Model


    Project Overview: Independently developed a football match prediction model that leverages historical data and statistical analysis to forecast match outcomes. This project aims to demonstrate the application of machine learning techniques in sports analytics, providing valuable insights for teams, analysts, and enthusiasts.


    Technology Used:
  • Machine Learning Algorithms: Implemented classification models such as Logistic Regression, Random Forest, and XGBoost to predict match results (win/draw/loss).
  • Data Collection & Preprocessing: Aggregated historical match data, player statistics, team form, and other performance metrics.
  • Feature Engineering: Created custom features like recent goal averages, head-to-head results, and home vs. away performance trends.
  • Evaluation Metrics: Assessed model accuracy using cross-validation and confusion matrices to refine prediction reliability.

    • Functionality:
  • Inputs match-specific data including recent form, team stats, and player performance.
  • Outputs probabilities for each possible match outcome.
  • Provides visual insights to help interpret factors influencing predictions.

    • Project Significance: This project highlights the use of data-driven approaches to enhance decision-making in sports. It demonstrates proficiency in data science, machine learning, and domain-specific feature engineering, showcasing the potential of predictive analytics in the sports industry.
